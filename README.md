# Blogging Platform

Full-stack blogging platform built with Node.js, Express, MongoDB (Mongoose) and a vanilla HTML/CSS/JS frontend.

Getting started:

1. Clone the repo
2. Copy `.env.example` to `.env` and set MONGODB_URI and JWT_SECRET
3. npm install
4. npm run dev

Features:
- User sign up / login (JWT)
- Create / edit / delete posts
- Comments
- Categories
- Search
- User profile
- Responsive frontend
